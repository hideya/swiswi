Swiswi
swift window switching
by swiping

Download

Precompiled and ZIP'ed (freeware)   for Mac OS X 10.9 or later

How to Run

  1. Unzip the downloaded zip file.
  2. Right click on the app icon, then select "Open", if you encounter some security error when double-clicking the app when running it.

Usage

Once started, move the mouse cursor onto the top menu bar or the desktop background, and make two-finger swipe on the track pad.

A vertical move switches the current window repeatedly just like pressing Cntl-F4.

A horizontal move switches the current tab repeatedly just like pressing Command-Shift-[.

To change the sensitivity of the switching, please click the icon on the status bar at the top of the screen and use the slider.
