Swiswi
swift window switching
by swiping

Download

Precompiled and ZIP'ed (freeware)   for Mac OS X 10.9 or later

How to Run

  1. Unzip the downloaded zip file.
  2. Right click on the app icon, then select "Open", if you encounter some security error when double-clicking the app when running it.

Usage

Once started, move the mouse cursor onto the top menu bar or the desktop background, and make two-finger swipe on the touch pad. The current window will switch repeatedly just like pressing Cntl-F4.

Press Cntl or Shift key together with the swipe for the Cntl-Tab action. This allows quick switching between browser tabs at the same time switching windows.

To change the sensitivity of the switching, pleae click the icon on the status bar at the top of the screen.
